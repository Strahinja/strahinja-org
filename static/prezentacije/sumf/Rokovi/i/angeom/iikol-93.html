3. Naci koordinate projekcije tacke M(5, 0, - 3, 4) cetvorodimenzionog prostora:

a) na hiperravan x1 + x2 - x3 + 2x4 = 2, ako je projekcijski zrak paralelan pravoj
x1 = 1 - t, x2 = 3 + 4t, x3 = 3t, x4 = 1 + t, t Î R;

b) na dvodimenzionu ravan
G : x1 - x2 + x3 + 1 = 0, x1 + x2 = x4 ako je
P : x1 + x2 + x3 + x4 = 0, x1 - 2x4 - 3 = 0 projekcijska dvodimenziona ravan.
